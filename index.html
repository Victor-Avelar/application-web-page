LED Keyboard Demo

Coding things like the tower defense game are helped tremendously by my favorite concept in the world: “open source”. Reading free documentation and implementing code others have built and then writing your own code on top of that is humbling. As a recent example: I acquired a programmable mechanical keyboard with back-lit led lights from Massdrop. With just a couple of hours of reading the free online documentation, I could write and compile code to fully customize the keymap and, most importantly, make custom LED animations.
